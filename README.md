# Angular Portfolio

This is a portfolio website converted from React to Angular 18 with Angular Material UI.

## Features

- Angular 18 with standalone components
- Angular Material UI
- Responsive design
- Routing with lazy loading
- TypeScript
- SCSS styling

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

## Build

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

## Project Structure

- `/src/app/components` - All Angular components
- `/src/assets` - Static assets (images, PDFs, etc.)
- `/src/styles.scss` - Global styles

## Components

- Home - Landing page with introduction
- About - About section with tech stack
- Projects - Portfolio projects showcase
- Resume - Resume viewer
- Navbar - Navigation bar
- Footer - Footer with social links

## Running the Application

```bash
cd portfolio-angular
npm install
ng serve
```

Then open http://localhost:4200 in your browser.
